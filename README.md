# SITE MANGA

## Getting Started

### Prerequisites

 * npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

1. Clone the repo
   ```sh
   git clone git@github.com:MohaGons/github-pages-project.git
   ```
3. Install NPM packages
   ```sh
   npm install
   ```
4. Run the project
   ```sh
   npm run start-dev
